背景

  • 把空白行殺掉

    • 問題是,linux 的換行是\n

    • 底下可以把文字檔顯示在文件中

      <pre>
{{*< trim  (readFile $filepath) "\r\n" | safeHTML >*}}
</pre>

  • 利用hugo 提供的函數chomp

    • 範例

      <pre>
{{*<   chomp (readFile $filepath) | safeHTML >*}}
</pre>

讀文字檔的內容,並顯示在文件上

將另一個文字檔,顯示在本文中,並且將裡面的空白行殺掉。這裡用到2個檔案

  • test1.html (short code) 💣 test1.html 的內容已經改了做其他測試。
  • index.md (測試內容)

測試用輸入_index.md ,最後有4個空白行。

---
title: Sata
description: stata 和HUGO之間的互動測試
weight: 2
---

  • 短碼內容 test1.html

    
<pre>
{{   trim  (readFile $filepath) "\r\n" | safeHTML}}
</pre>
by chomp <br/>
<pre>
{{   chomp (readFile $filepath) | safeHTML}}
</pre>
by replaceRE <br/>
<pre>
{{   replaceRE "\\s*$" "" (readFile $filepath)  | safeHTML}}
</pre>
{{- print "" -}}

weight: 2
