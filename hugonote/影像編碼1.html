single.html -> calls partial.html
Within single.html {{ range .Resources }} is used, which then provides data to the partial {{ partial "partial.html" . }}.

A few issues come up then.

.Name is not the name of the resource file, but of the calling index.md file
.Resources.GetMatch "*" doesn't return a result
$.Site.Resources or $.Resources doesn't work
Current workaround using scratch:

single.html:
{{ range sort .Resources }}
{{ $.Scratch.Set "data" . }}
{{ partial "fragments/test.html" $ }}
{{ end }}

fragments/test.html:
{{ $self := ($.Scratch.Get "data").Params }}
{{ $name := ($.Scratch.Get "data").Name }}
{{ $name_t := strings.TrimSuffix ".md" $name }}
{{ $content := .Resources.Match (printf "%s/*" $name_t) }}
{{ range $content }}
{{ .Params }}
{{ end }}

如何下載URI

encodeimg2短碼 
{{- $pattern := default "[.]*\\.(gif|jpg|jpeg|tiff|png|bmp)" (.Get "pattern") -}}
{{- $prompt:=default "嵌入檔案"  (.Get "title") -}}
{{- $section := default "report" (.Get "section") -}}
{{- $page:= default (.Site.GetPage "section" $section) -}}


{{- range  $page.Resources  -}}
{{- $filetype := .MediaType.SubType -}}
{{- $webpath := .RelPermalink -}}
{{- $path := path.Join $page.Dir .Name -}}
{{- $s64:= base64Encode .Content -}}
{{- if findRE $pattern .Name -}}

{{- if  (findRE "(gif|jpg|jpeg|tiff|png|bmp)" $filetype) -}}
<img class="embeded" style="width:50%" src="data:image/{{- $filetype -}};base64,{{- $s64 -}}"> 
{{- else -}}
<i class="fas fa-paperclip" aria-hidden="true"></i>
<a download="{{.Name}}" href="data:@file/{{$filetype}};base64,{{- $s64 -}}">{{- $prompt -}}: {{.Name}}</a><br/>
{{- end -}}

{{- end -}}

{{- end -}}
