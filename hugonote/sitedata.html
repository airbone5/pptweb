  • 怪怪,好像不是很正確的參考

目的

怎樣存取.Site.Data 裡面的json

實驗和材料

├─content
│  └─blogs
|        └─sitedata.md 🔑 執行短碼whatdata(.html)的文件,實驗時候是本文。
├─data
│  │
│  ├─xxx
|  │  └─xx.toml  🔑實驗要存取的對象
|  └──x2.toml 🔑實驗要存取的對象
├─resources
├─static
└─themes
  • 簡單記憶的話,就是.Site.Pages 是content 子目錄下的檔案,而.Site.Data就是Data子目錄下的檔案。
    • 所以,這個data子目錄,和content是平行的,如上圖。
    • 在這個子目錄下可以有多個yml 或toml 每個yml 如下
      yml
      ```yml name: Leo Tolstoy bio: "Russian novelist, best known for his novels *War and Peace*, *Anna Karenina* and various essays." sources: - title: "Anna Karenina" quotes: - entry: "quote no. 1" comment: "comment no. 1" credits: "credit no. 1" - entry: "quote no. 2" comment: "comment no. 2" credits: "credit no. 2" - title: "War and Peace" quotes: - entry: "quote no. 3" comment: "comment no. 3" credits: "credit no. 3" ```
      json
      如果是json ``` { "name": "Leo Tolstoy", "bio": "Russian novelist, best known for his novels *War and Peace*, *Anna Karenina* and various essays.", "sources": [ { "title": "Anna Karenina", "quotes": [ { "entry": "quote no. 1", "comment": "comment no. 1", "credits": "credit no. 1" }, { "entry": "quote no. 2", "comment": "comment no. 2", "credits": "credit no. 2" } ] }, { "title": "War and Peace", "quotes": [ { "entry": "quote no. 3", "comment": "comment no. 3", "credits": "credit no. 3" } ] } ] } ```
      toml
      如果是toml ```toml name = "Leo Tolstoy" bio = "Russian novelist, best known for his novels *War and Peace*, *Anna Karenina* and various essays." [[sources]] title = "Anna Karenina" [[sources.quotes]] entry = "quote no. 1" comment = "comment no. 1" credits = "credit no. 1" [[sources.quotes]] entry = "quote no. 2" comment = "comment no. 2" credits = "credit no. 2" [[sources]] title = "War and Peace" [[sources.quotes]] entry = "quote no. 3" comment = "comment no. 3" credits = "credit no. 3" ```

  • 存取資料檔的辦法,例如: ./data/novelists/authors/*.yml 可以由 .Site.Data.novelists.authors存取。

    • {{ range .Site.Data.novelists.authors }}
<h3>{{ .name }}</h3>
<p>{{ .bio }}</p>
{{ range .sources }}
<h4>{{ .title }}</h4>
{{ range .quotes }}
  <blockquote>
    <p>{{ .entry }}</p>
    {{ if .credits }}<p><small><em>{{ .credits }}</em></small></p>{{ end }}
  </blockquote>
  {{ if .comment }}<p>{{ .comment }}</p>{{ end }}
{{ end }}
{{ end }}  
{{ end }}
