    自訂輸出

    我想有一個獨立的網頁,例如內容是所有文章的列表,當開啟單獨一個網頁的時候,就引用內容,而不是每次都在每個網頁包含全部的列表。

    來自官網的節錄

    Example from site config file:

    config.toml

    [outputs]
          home = ["HTML", "AMP", "RSS"]
          page = ["HTML"]
    • .Page.Kind 如果是page ,那麼就輸出HTML。
    • 上面的設定 不影響內定(例如.Kind為 section和 taxonomy) ,兩者仍然是輸出 [“HTML”, “RSS”].

    我的實驗

    總之就是,home除了產生index.JSON以外,現在又用來產生⛳walk.html這個裡面就只是一堆文章的ul,li 列表。

    而產生前述的列表,就像產生index.json一樣,需要模板,也就是home.walk.html,而在這個模板中,調用partial/walk.html的遞迴函數。

    最後,在site/index.html中,需要將上面的⛳動態插入到partial/menu.html中的div#side-walk。

    我偷懶,照下面的流程做下去,除了HOME以外,其他的側欄都是空的,因為,我沒有在single.html,list.html 中加入最後一個步驟的JS。

    1. config.toml 的原來設定只有

      [outputs]
        home = [ "HTML", "RSS", "JSON"]

      上面只對HOME網頁產生兩個輸出,一個是docs/index.html,一個是docs/index.json

      現在打算多產生一個sidebar.html,改成如下:

      [outputFormats]
        [outputFormats.walk]
        mediaType = "text/html"
        baseName = "walk" # 
      🏷 在根目錄下產生walk.html
      
        isHTML = true
        #isPlainText = true
        notAlternative = true
        [outputs]
        home = [ "HTML", "RSS", "JSON","walk"]

      homepage 除了index.html 還會有walk.html,index.json。 RSS不產生,因為沒有對應的模板

    2. 建立檔案layouts\home.walk.html作為產生walk.html的模板,內容如下:

      <ul>
        {{ range where .Site.Pages "Kind" "section" }}
          <li>
            <a href="{{ .RelPermalink }}">{{ .Title }}</a>
            {{ if .Pages }}
              {{ partial "walk.html" . }}
            {{ end }}    
          </li>
        {{ end }}
        </ul>
      • 這是配合config.toml中的自訂輸出
      • 只是遞迴調用layouts/partial/walk.html:

    3. 用來產生sidebar的遞迴(給layouts\home.walk.html調用的) layouts/partial/walk.html的內容 

      <ul> 
        {{ range .Pages.ByTitle }}
          <li>
            <a href="{{ .RelPermalink }}">{{ .Title }}</a>
            {{ if .Pages }}
              {{ partial "walk.html" . }}
            {{ end }}    
          </li>
        {{ end }}
        </ul>

    4. layouts/partial/menu.html,只是把variant/layouts/partial/menu.html複製來這裡,然後把其中的template殺除,只放入一個空的<div id="side-walk">

        {{ $currentNode := . }}

            <section id="homelinks">

                <li>
                    <a class="padding" href='{{ (cond (ne .Site.Params.landingPageURL nil) .Site.Params.landingPageURL "/") }}'>{{ safeHTML (cond (ne .Site.Params.landingPageName nil) .Site.Params.landingPageName (printf "%s%s" "<i class='fas fa-home'></i>" .Site.Title )) }}</a>
                </li>
              </ul>
            </section>
          {{end}}   
            {{if not .Site.Params.disableSearch}}

                 {{/* .Render "summary" */}}
