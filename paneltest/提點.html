但是有flex的情況下,被flex優先置換。

flex

(實際範例)[https://ithelp.ithome.com.tw/articles/10193306]： - flex-grow: 延伸空間 = (該元素flex-grow/所有元素flex-grow總和)*剩餘空間 - flex-shrink: 壓縮空間 = -(該元素flex-shrink*flex-basis/所有元素flex-shrink*flex-basis總和)*剩餘空間 原本每個元素的寬為200px，但因為flex容器的大小為1000px，所以剩餘空間為1000-8x200 = -600px，而每個元素flex-shrink預設值都為1，且flex-basis = 100px，因此每個元素的壓縮空間都為 (1x100/8x1x100)x600，因此壓縮後每個元素的大小都為200-75=125px。

如果flex 有隱藏元件,最好在容器之前先宣告display:none否則,flex會把它排版,然後消失的時候,又排版,閃2次。

bootstrap

For aligning within a flexbox div or row...

  • ml-auto is now ms-auto
  • mr-auto is now me-auto

For text align or floats..

  • text-left is now text-start
  • text-right is now text-end
  • float-left is now float-start
  • float-right is now float-end

bootstrap accordion

  • class="accordion": 最外層
  • data-bs-toggle="collapse": collapse
  • data-bs-target: 誰要被控制
  • data-bs-parent: 沒有也可以,指向最外層的id推測是按了其中一個按鈕,其他就被關掉。
