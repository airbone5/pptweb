插入github 上的程式碼

顯示github上的程式碼 ,可以利用短碼

⛳範例(來自)

例如要插入的檔案在https://github.com/airbone5/blogtest/blob/master/content/post/2015-01-01-lorem-ipsum.md,則指令如下

{{< github repo="airbone5/blogtest" file="/content/post/2015-01-01-lorem-ipsum.md" lang="md" >}}

參數只有repo和 file 參看下圖 短碼github.files/filelocation.png

more

執行結果 >

檔案/content/post/2015-01-01-lorem-ipsum.md:內容

另一個範例

browser網址欄位上顯示的是 https://github.com/KevinGimbel/hugo-kg-shortcodes/blob/main/layouts/shortcodes/codepen.html 但是使用github 短碼的時候,這個路徑名稱不能用(參考上圖的❌)

而是{{< github repo="KevinGimbel/hugo-kg-shortcodes" file="layouts/shortcodes/codepen.html" >}}

其執行結果如下:

檔案layouts/shortcodes/codepen.html:內容

{{ .Page.Scratch.Set "kg_include_codepen" true }}

{{ $height := .Get "height" | default "256" }}
{{ $theme := .Get "theme" | default "dark" }}
{{ $result := .Get "result" | default "html,result" }}
{{ $user := .Get "user" | default .Site.Params.kg_shortcodes.Codepen.User }}
{{ $editable := .Get "editable" | default .Site.Params.kg_shortcodes.Codepen.Editable }}
{{ $preview := .Get "preview" | default .Site.Params.kg_shortcodes.Codepen.Preview }}
<p data-height="{{ $height }}" data-theme-id="{{ $theme }}" data-slug-hash="{{ .Get "slug" }}" data-default-tab="{{ $result }}" data-user="{{ $user }}" data-embed-version="2" data-pen-title="{{ .Get "title"}}" data-preview="{{ $preview }}" data-editable="{{ $editable }}" class="codepen">
    See the Pen <a href="https://codepen.io/kevingimbel/pen/{{ .Get "slug" }}/">{{ .Get "title" }}</a> on <a href="https://codepen.io">CodePen</a>.
</p>
