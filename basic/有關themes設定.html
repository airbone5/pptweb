TOML JSON 的對照範例 config.toml 的設定

背景

這裡的config 用的是toml xxxxx - 本地toml 的參考

baseURL

最重要的設定是 #第一個不要斜線,最後一個要斜線,如下

baseURL: "/" #如果不需要路徑,這樣就可以,否則例如要掛到ftp
baseURL: "帳號/project/nn/" #第一個不要斜線,最後一個要斜線
baseURL 如果不是設定成/,則relativeURL 必須是false。否則會重複出現在連結路徑中。

uglyURL

menu設定

[[menu.shortcuts]] 
name = "<i class='fab fa-github'></i> Github repo"
identifier = "ds"
url = "https://github.com/matcornic/hugo-theme-learn"
weight = 10

分塊的寫法,示範怎樣拿取上面的值。

  {{with .Site.Menus.shortcuts}}
  <section id="shortcuts">
    <h3>{{ if not $disableShortcutsTitle}}{{ T "Shortcuts-Title"}}{{ end }}</h3>
    <ul>
      {{ range sort . "Weight"}}
      <li>
        {{.Pre}}<a class="padding" href="{{.URL | absURL }}">{{safeHTML .Name}}</a>{{.Post}}
      </li>
      {{end}}
    </ul>
  </section>
  {{end}}
  • .Site.Menus 這是HUGO的內定參數。
