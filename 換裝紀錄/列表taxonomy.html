總覽

列出標誌文章的模板來源(參考):

layouts
│
├─partials
├─shortcodes
├─categories (📌但是當時實驗的時候,是放在專案中的layout而不是themes中)
│      🏷taxonomy.html (==category.terms.html,只是這裡優先)
│
├─pages 🏷如果存在`content/pages`
│      🏷single.html 優先取代_default中的single.html
│      🏷list.html 優先取代_default中的list.html
│      
└─_default
        category.terms.html 🏷列出categories中,被標示的文章
        list.html 🏷terms.html,category.terms.html都不存在時,就用這個。
        single.html
        terms.html 🏷每個taxonomy 的對應(e.g. category.terms.html) 不存在時,就用這個)列出被標示的文章

兩種taxonomies,以categories為例,一種列出所有在這個categories下所有的詞條(模板layouts/categories/term.html),一種列出每個詞條有那些文章(模板layouts/categories/taxonomy.html),兩種都可以使用_default/list.html。

以categories為例() - terms pages(e.g. /_default/terms.html) - Category terms page(例如 categories: ["x","y"] : 裡面的x,y 都會被列表) - list of taxonomies - 程式碼: {{ range .Data.Terms.ByCount }} - taxonomy pages (e.g., /_default/taxonomy.html) - Category taxonomy page(只對其中一個(如x),類似section文章列表) - one taxonomy item - 程式碼會用到: {{range .Pages}}(💡因為可以不要這個,就只需要_default/list.html) 或 {{range .Data.Pages}} 或 {{ range .Paginator.Pages }}

  • ✔如果不是在taxonomy模板,例如shortcode 或 partial 則要利用.Site.Taxonomies 或 .Site.Page + .Kind 來判別
  • 如果不是因為uglyURL 上面兩個都不用。但是因為,所以至少需要terms.html。
    • 因為設為真的時候,HUGO不知道要去找有HTML結尾的網址。列如對所有category上的terms(e.g. categories: ["x","y"])列表,預設是去找http://site/categories/x,但是我們要http://site/categories/x.html

💯

│  
├─分類(所有詞條)terms pages(模板list.html))   
      └──個個詞條taxonomy pages(對應section,模板list.html)  
          └──個別文章(模板single.html)

[[menu.main]]

如何有不同的layout

如果不同的section(例如content/pages) 要有不同的layout 那就可以在layouts\pages在新增相對的single.html和list.html(不是❌layouts\_default\pages)

Q&A

.Site.Pages vs .Data.Pages

  • .Site.Pages = All the Pages in the site.
  • .Data.Pages: All the Pages for the given Node (sections, taxonomies). This will be a subset of the above for all other nodes than the home page: On that Node I believe these slices are the same.

tags 列表

短碼 listtags.html 的內容(所以利用的是.Site.Taxonomies)

{{- range $name, $taxonomy := .Site.Taxonomies.tags -}} 
    {{- $cnt := .Count -}}
    {{- with $.Site.GetPage (printf "/tags/%s" $name) -}}
      <div class="tagbutton">
		<a href={{ .RelPermalink }} title="All pages with tag {{$name}}">{{$name}}</a>
		<sup>{{$cnt}}</sup>
      </div>
	  {{- end -}}
{{- end -}}
