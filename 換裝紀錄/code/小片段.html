{{end}}
結果
由下面傳回的資料(節錄)有home,section,page,taxonomy,term 。 ------------- HUGO的相關測試 , section test , home test hugo variable , page basic , term Categories , taxonomy Hugo , term Series , taxonomy Tags , taxonomy taxonomy , term

❌{{.Data.Pages}} 出錯

拿所有的taxonomy

利用where

{{ range where .Site.Pages "Kind" "taxonomy" }}
{{ .Name }} 🐞
{{ .Relpermalink }}
{{ .Permalink  }}
{{end}}

結果 Categories Series Tags ✔config.toml 有這三個taxonomy

.Data.Terms.

    {{ $data := .Data }}
	{{ range $key,$value := .Data.Terms.Alphabetical }}
	  <li class="site-tag"> {{ $data.Plural }}, {{ $value.Name }} 🐞 </li>
	{{ end }}

結果

categories, hugo 🐞 🏷前面是.Data.Plural,後面是$value, $key 只是號碼。
categories, stata 🐞

這樣也可以列出所有的tax


{{ range where .Site.Pages "Kind" "taxonomy" }}
{{ .Title }} ,  {{.Kind}} {{.Permalink }} <br/>
{{end}}

抽出參數含有某些值的網頁

{{ range where .Site.Pages ".Params.tags" "intersect" .Params.tags }}
  {{ if ne .Permalink $.Permalink }}
    {{ .Render "summary" }}
  {{ end }}
{{ end }}
