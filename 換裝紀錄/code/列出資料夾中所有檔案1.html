利用.GetPage

執行指令範例: {{< listsection section="/pages/test" >}}

{{ $section := default "report" (.Get "section") }}
<ul >
   {{/* 
      這是舊的寫法* , .Site.GetPage $section 就可以.
   /}}
   {{ $apage :=.Site.GetPage "section" $section}} 
   {{ range $apage.Pages}}
     <li>{{ .Title }} </li>
   {{ end }}  
</ul>

.GetPage:傳回一個網頁而不是Section 下的所有網頁。使用範例為.Site.GetPage (/web)(拿到的內容為Page(/web/_index.md))。

例如上例$apage(就是說.Kind=="section"),然後可以用函數$apage.Pages 可以得到其他MD,同理,tags也是這樣,參考下一節。

修改上面的迴圈,只是找出附檔名

{{ $list := .Scratch.Get "list" }}
{{ range $list  }}

{{ end }}
